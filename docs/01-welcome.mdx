# Welcome to the VotV Modding Wiki!

Welcome! This wiki will explain all about how to:
- Install mods
- Create your own mod
- And how to troubleshoot any issues you might find

You can click on any tab on the left sidebar to jump to the related documentation page, or you can use the Next/Previous buttons on the bottom of each page.

This documentation is maintained by Questwalker (@questwalker on Discord) and the modding community, report any issues in the EternityDev Discord.

If you want to contribute, check out the contributing directions on [here](https://github.com/Questwalker/votv-modding-wiki?tab=readme-ov-file#contributing)! Any kind of contributions are greatly appreciated!
