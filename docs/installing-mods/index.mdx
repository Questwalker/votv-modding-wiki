# Installing Mods
These directions are for installing and running mods for VotV.

There are two general options:
* Using a mod manager
* Installing everything manually

Since mod managers help automate everything for you, including installing the correct files to the correct locations and disabling/enabling mods at your lesuire, its heavily recommended to use a mod manager.
